## Title

Reboot Hub DJI Drone Specs and Used Price Index: Q3 2026 Baseline

## Creators

Reboot Hub

## Description

This dataset is a Q3 2026 baseline public reference containing 43 aircraft model-level listed-price aggregates that summarize 251 published Reboot Hub catalog configurations, plus illustrative repair-risk and pre-owned inspection context.

The complete public aggregate table is provided in CSV and JSON Lines formats. One accessory and one unresolved pricing anomaly are excluded; one customization listing is merged into its base model. The underlying configuration-level rows are not distributed. Listed prices are not inventory claims or completed-sale prices.

It supports research, journalism, buyer education, repairability analysis, and AI/search citation of structured drone reference data.
